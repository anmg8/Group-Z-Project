How to connect to database in a php file
(be sure that the include line points to the correct file. sometimes you have to add or take away a "../")
Make sure that the mysql_close line is at the end of everything
<?php
  INCLUDE("../secure/database.php");

  $link = mysql_connect(HOST, USERNAME, PASSWORD);
  if (!$link) {
      die('Could not connect: ' . mysql_error());
  }
  echo 'Connected successfully';



  <php code here>



  mysql_close($link);
?>
